# CapWeaver

CapWeaver is a lightweight coding agent project focused on **capability formation**.

The codebase keeps the small and readable structure inherited from CoreCoder, and extends it with a controlled lifecycle for reusable capabilities:

- search existing skills first
- forge a temporary tool when needed
- decide whether that tool should be discarded, kept for the current session, or promoted into a persistent skill
